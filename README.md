# テーマ1　テーマ「日田市の魅力についてまとめる」

## 概要
日田市の魅力をテーマに制作する。豊かな自然や文化、人々の暮らしに着目し、それらが調和することで生まれる地域の魅力を表現する。

### ページ１

日田市の基本情報についてまとめます。
所在地
大分県西部に位置する市。福岡県・熊本県と接し、北部九州のほぼ中央に位置する交通の要衝。

面積
約666.03㎢。市域の大部分を森林が占め、豊かな自然環境に恵まれている。

人口
約5.9万人（2026年5～6月時点）。世帯数は約2.8万世帯

### ページ２
日田市の文化や観光についてまとめます

天領文化、小鹿田焼、日田祇園祭、ひな文化など

### ページ３
日田の食と特産品についてまとめます

日田焼きそば、鮎料理・ヤマメ料理、ひたん寿司など
特に日田焼きそばは日田市を代表するソウルフードで外はパリっと中はもちもちとした食感なのでとてもおすすめです

日田市の歴史ある街並みについてまとめます
